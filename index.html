OzariasValidation

jQuery.ozariasValidation v1.1

Validação de formulario de forma simples e facil.

Requisitos necessários

<script src="https://ajax.googleapis.com/ajax/libs/jquery/1.12.4/jquery.min.js" type="text/javascript"></script>
<script src="js/ozariasValidation-1.1.js" type="text/javascript"></script>

Exemplo de uso

<script type="text/javascript">
$(document).ready(function(){
    $("#exmplo1 button").click(function(){
        if ($("#validatorExemplo1").ozariasValidation().validateForm()) {
            alert("Formulário válido.");
        }
    })
})
</script>

Faça o download clicando aqui.


Valida campos obrigatórios, expressões regulares, funções personalizadas, valor mínimo, valor máximo, valor inteiro, minlength, maxlength, campos de confirmação, e por ajax.

Para escolher o tipo de validação do campo, basta colocar na classe do campo

Classes de validação

  • class="validate[required]"
  • class="validate[confirm[email]]"
  • class="validate[min[5]]"
  • class="validate[max[5]]"
  • class="validate[minlength[10]]"
  • class="validate[maxlength[10]]"
  • class="validate[int]"
  • class="validate[ajax[url]]"
  • class="validate[func[nomeFuncao]]"

Parametros de configuração

  • language: 'pt-br',//string 'en', 'es', 'fr', 'it'
  • divErrorPosition:"bottom",//string top or bottom default:bottom
  • divErrorContent: divErrorContent,// div object
  • completeValidateField: function(elementField, success),
  • completeValidateForm: function(elementForm, fieldErrors, object)
